## 📁 Como baixar o projeto

  ```bash
  # Clonar o repositório
  $ git clone https://github.com/douglasbrgs/csharp-completo
  # Entrar no diretório
  $ cd csharp-completo
  # Iniciar o projeto
  $ dotnet run
  ```
